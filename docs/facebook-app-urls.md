## Important Notes

⚠️ **ngrok Free Limitations:**
- The URL changes every time you restart ngrok
- Free tier has session limits
- For production, use a stable domain

⚠️ **For Facebook App Review:**
- Facebook reviewers need access to these URLs
- Make sure ngrok is running during review
- Consider using ngrok paid plan for stable URL
- Or deploy to a production server with a real domain

## Alternative: Production URLs
